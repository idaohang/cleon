DESCRIPTION

  • The firmware has been developed in collaboration between ESL(Embedded Systems Lab) at ECE, Purdue Univ and SERG(Sensing and Energy Research Group) at Microsoft Research, Redmond
  • The MCU used in this platform is MSP430F5338IPZ
    (http://www.ti.com/product/msp430f5338)
  • FatFs file system is used for MicroSD card interface
    (http://elm-chan.org/fsw/ff/00index_e.html)
  • Parameters should be updated by PC side parameter updating SW named 'CLEON CONNECTOR'
    (http://woosuklee.github.io/cleon_connector)
